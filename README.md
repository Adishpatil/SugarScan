# ?? SugarScan — AI Sugarcane Disease Detection

A Flutter mobile app that detects sugarcane diseases using deep learning.

## Features
- ?? Camera & Gallery image input
- ?? AI detection using EfficientNetB0 TFLite model
- ?? Detects 5 conditions: Healthy, Mosaic, Rust, Red Rot, Yellow Leaf
- ?? Confidence scores for all classes
- ?? Scan history
- ?? Disease info with treatment guidance- 100% offline
## Model
- Architecture: EfficientNetB0
- Dataset: 5 class sugarcane leaf dataset
- Input: 224x224 RGB images

## Built With
- Flutter 3.19.6
- TFLite Flutter 0.12.0
- Hive (local storage)
